## 🚀 Quick Start

Follow these steps to get the project running locally.

### 1. Prerequisites
* Node.js (v18 or higher)
* npm or yarn

### 2. Installation
Clone the repository and install the dependencies:
```bash
git clone https://github.com
cd your-repo-name
npm install
```

### 3. Environment Setup
1. Copy the template file:
   ```bash
   cp .env.example .env
   ```
2. Open `.env` and add your specific API keys and secrets.

### 4. Run the App
```bash
npm run dev
```
